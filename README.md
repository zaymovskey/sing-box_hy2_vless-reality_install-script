# 🚀 sing-box One-Click Install

Скрипт для быстрой установки и настройки **sing-box** с поддержкой:

- VLESS Reality (TCP, 443)
- Hysteria2 (UDP, 8443)

Полностью автоматическая установка:
- установка sing-box
- генерация ключей (Reality)
- генерация UUID
- генерация TLS сертификата (self-signed)
- создание конфига
- настройка systemd
- запуск сервиса
- вывод готовых клиентских ссылок

---

## ⚡ Установка

```bash
chmod +x install.sh
./install.sh
